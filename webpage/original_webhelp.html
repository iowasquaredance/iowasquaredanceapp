Iowa Square Dance Schedule

This is how to install the Android phone app containing a list of square dances in Iowa. The main schedule screen looks like this...

You may attempt download and installation without reading the instructions below BUT expect a request to permit installation of apps from "unknown sources". Recommendation: give permission but once the app is working, return to settings and turn the permissions back off.

Download app

After clicking the "Download" button, you might be asked which browser you want to use. Expect this if you have not yet used the browser app on your phone. Chrome is OK but remember which one you choose.

Look for an option to "open".
